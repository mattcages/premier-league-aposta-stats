# Football Data Scraper Suite

A complete Python-based automation suite that collects and organizes football data from SofaScore, including clubs, matches, and players.

## Overview

This project is designed to scrape and structure football data into a clean and organized local database.

It includes:

- Club statistics scraper  
- Match data & reports generator  
- Player statistics extractor  

All data is stored in a structured directory for easy access and analysis.

---

## Technologies

- Python 3
- Selenium
- GeckoDriver (Firefox)
- WebDriver Manager
- JSON
